## Assets

Test assets
```bash
  bundle exec rake assets:clobber
  ASSETS=true NG_FORCE=true RAILS_ENV=development bundle exec rake assets:precompile
  ASSETS=true rails s -p 3010
```

Cleanup
```bash
  bundle exec rake assets:clobber
```
